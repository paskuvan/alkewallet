# Alke Wallet - Billetera Digital

## Autor
**Maria Jose Paschcuan**

## Descripción
Alke Wallet es una aplicación de billetera digital que permite a los usuarios gestionar sus transacciones financieras de forma segura y conveniente.

## Características
- Login seguro
- Depósitos
- Envío de dinero
- Historial de transacciones
- Menú de navegación intuitivo

## Archivos del Proyecto
- `index.html` - Página de inicio
- `login.html` - Formulario de login
- `menu.html` - Menú principal
- `deposit.html` - Funcionalidad de depósitos
- `sendmoney.html` - Funcionalidad para enviar dinero
- `transactions.html` - Historial de transacciones
- `style.css` - Estilos de la aplicación

## Instrucciones de Uso
1. Abre `index.html` en tu navegador
2. Inicia sesión con tus credenciales (admin - 1234)
3. Accede a las diferentes opciones desde el menú
4. Realiza depósitos o envía dinero según sea necesario
